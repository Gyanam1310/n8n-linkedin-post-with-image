# 🚀 n8n AI LinkedIn Post with Image

An end-to-end automation workflow that converts any article into a **viral-ready LinkedIn post + AI-generated image** using **n8n**.

---

## 🔥 Overview

This project automates the entire content creation pipeline:

* 📰 Takes an article link from Google Sheets
* 🧠 Summarizes it using AI
* ✍️ Generates a high-impact LinkedIn post
* 🎨 Creates an AI image prompt
* 🖼️ Generates and downloads an image
* 📤 Publishes directly to LinkedIn

---

## ⚙️ Tech Stack

* **n8n** – Workflow automation
* **Google Sheets** – Input source
* **Google Gemini / LLM APIs** – Content generation
* **Freepik AI API** – Image generation
* **LinkedIn API** – Auto posting

---

## 🧠 Workflow Architecture

```text
Google Sheets → AI Summary → LinkedIn Post → Image Prompt → Image Generation → Download → LinkedIn Post
```

## 🔐 Setup Required

Before running the workflow, configure:

* ✅ Google Sheets credentials
* ✅ Gemini / LLM API key
* ✅ Freepik API key
* ✅ LinkedIn OAuth credentials

Replace placeholders in workflow:

```
YOUR_API_KEY
YOUR_GOOGLE_SHEET_ID
YOUR_STATUS_URL
```
